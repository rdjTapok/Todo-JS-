## ✨ Features

- ✅ Add/delete tasks with intuitive interface
- 🟢 Mark tasks as complete/incomplete
- 🔍 Filter tasks by status (all/active/completed)
- 📅 Set due dates for tasks
- 💾 Local storage persistence
- 📱 Responsive design
